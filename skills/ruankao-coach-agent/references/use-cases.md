# 软考教练通用用例库

这些用例是开源版通用流程，不依赖任何私人资料路径或固定题目。

## 用例 1：综合题/选择题错题闭环

触发语：`这道综合题错了`、`把这题记下来`、`选择题错题复盘`

自动动作：

1. 用 `ruankao-wrong-note` 记录题干、选项、用户答案、正确答案和错误原因。
2. 挂到 `软考/输出/知识点索引.md` 的可复习知识点。
3. 当场追问 1 个最小记忆点，确认不是只会看解析。

## 用例 2：案例模拟题做完后闭环复盘

触发语：`帮我批改这套案例`、`案例模拟题做完了`、`做完按采分点复盘`

自动动作：

1. 用 `ruankao-answer-grader` 批阅答卷。
2. 汇总 3 个主要失分点。
3. 只记录真正属于知识点缺口的错题。
4. 用 `ruankao-case-2sigma-review` 补最影响分数的 1 个知识点。

## 用例 3：案例知识点专项补洞

触发语：`我不会 CAP`、`质量属性场景再考我`、`补洞这个案例知识点`

自动动作：

1. 在用户上传的案例教材、真题或题库中检索知识点。
2. 出 1 个诊断题。
3. 根据回答判定 `3/3、2/3、1/3、0/3`。
4. 不完整时用 2sigma 小步追问直到达到基本掌握。

## 用例 4：案例冲刺抽查

触发语：`来一轮案例冲刺`、`随机抽 10 题`、`按章节抽案例`

自动动作：

1. 检查 `软考/输出/案例冲刺/题库.json`。
2. 如不存在，提示先上传 `软考/资料/案例题/案例教材.pdf` 和 `案例真题.pdf`，再运行 refresh。
3. 用 `ruankao-case-sprint-10` 抽题、判题和更新掌握状态。

## 用例 5：论文背诵与迁移

触发语：`复习论文`、`背这篇范文`、`这个母题怎么迁移`

自动动作：

1. 用 `ruankao-thesis-2sigma` 做三层训练：知识理解、范文骨架、子题迁移。
2. 只问 1 题，等用户回答后反馈。
3. 项目动作只从 `软考/资料/项目背景/项目背景卡.md` 中提取。

## 用例 6：论文写完后评审

触发语：`论文写完了帮我评审`、`这篇能不能过`、`按考官标准估分`

自动动作：

1. 用 `ruankao-thesis-reviewer` 给完整评分。
2. 提取 3-5 个最大扣分点。
3. 如果是项目实践空，用 `ruankao-thesis-argument-prep` 补一段项目动作。
4. 如果是表达重复，直接给替代表达。

## 用例 7：第一次使用

触发语：`我刚装好`、`资料怎么放`、`初始化软考技能`

自动动作：

1. 转入 `ruankao-init`。
2. 创建 `软考/资料` 与 `软考/输出` 目录。
3. 检查教材、真题、范文、项目背景卡是否齐全。
4. 给一个最小下一步，不一次安排过多任务。
